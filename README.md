# Site de Vente de Vêtements de Sport

## Description

Application web dynamique dédiée à la vente de vêtements de sport, développée avec HTML, CSS, JavaScript, PHP et MySQL.

## Fonctionnalités

* Génération dynamique des produits à partir des images locales
* Filtrage des produits par sport, genre et catégorie (AJAX, sans rechargement)
* Système d’authentification (inscription et connexion)
* Interface responsive et conviviale

## Remarque

* Les produits ne sont pas stockés en base de données, ils sont générés automatiquement à partir des images
* La base de données est utilisée uniquement pour la gestion des utilisateurs

## Auteur

Projet réalisé dans le cadre d’un TP de développement web.
